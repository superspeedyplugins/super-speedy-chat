How it works

When a visitor sends a message, the request is handled by the Ultra Ajax mu-plugin, which intercepts the call before the full WordPress stack loads — keeping response times in the tens of milliseconds with minimal server load.

Conversations are stored in custom database tables on your own server. Your admin dashboard and the visitor's widget each poll for new messages on their own adaptive interval, so chats feel instant without hammering your host.

If you enable Discord, every new conversation opens a thread in your channel, and a small companion bot relays your Discord replies back to the visitor through a secured endpoint — fully bidirectional, with no third-party middleman.

Frequently asked questions

Will it slow down my website?

No — the opposite of what most chat tools do. There are no heavy third-party scripts loading on your pages, and chat requests are handled by a must-use plugin that responds before the full WordPress stack boots, keeping response times in the tens of milliseconds. Your Core Web Vitals and TTFB stay intact.

Is it really free? What's the catch?

The core plugin is genuinely free and open-source — no nag screens, no conversation limits, no per-seat fees. It's the same engine we use ourselves. Optional paid add-ons exist for advanced needs (and fund ongoing development), but the vast majority of sites never need them.

What happens when a visitor messages and I'm not online?

You're covered three ways: visitors can leave their email so you can follow up (and your reply is emailed straight back to them), an optional AI can auto-answer common questions from your canned responses, and because you can reply from Discord, you can keep chatting from your phone wherever you are.

Do I need to be a developer to set it up?

No. Install, activate, and a chat bubble appears on your site — the database tables and speed-boosting mu-plugin install themselves. Colours, position, icon and wording are all set in the WordPress Customizer with a live preview. No code, no API keys for the core chat.

Will it work with my theme, page builder and caching plugin?

Yes. The widget loads through standard WordPress hooks, so it works with virtually any theme or builder, and it plays nicely with caching because the chat endpoints are dynamic and bypass the page cache. It's a great fit for WooCommerce stores that can't afford a slow site.

Where is my chat data stored? Is it GDPR-friendly?

Everything lives in custom tables in your own WordPress database — nothing is sent to a third party unless you switch on the optional Discord or AI features. Keeping conversations on your own server makes GDPR compliance far simpler than shipping customer data to an overseas SaaS.

Can my whole team use it?

Yes. Conversations appear in a shared wp-admin inbox where you can assign them to specific admins, filter by assignee, and search by visitor. Each agent can use their own display name or a single shared one like “Support”.

How does the Discord integration work, and is it hard to set up?

Each new conversation opens a thread in your Discord channel automatically. To reply from Discord, you run a small companion bot (one command on any Node.js server) that relays your messages back to the visitor in real time. Visitor-to-Discord works the moment you paste in your bot token and channel ID.

Does the chat work on mobile?

Yes — the bubble is fully responsive and adapts to the on-screen keyboard, so visitors get a clean experience on phones and tablets.

How do you stop spam and bots?

The chat includes a hidden honeypot field that silently rejects bots and built-in per-IP rate limiting on sending and session creation, so automated abuse is throttled out of the box. You can also require login before chatting.

Does the AI auto-reply cost anything?

The feature is optional and uses your own OpenAI or Anthropic key with their cheapest models, so each answer costs a fraction of a cent. There's a built-in daily cap to keep spend predictable, and you can leave it switched off entirely.

What if I get stuck?

There's a full knowledge base covering setup, configuration, and the Discord and AI features, and priority support is available as an add-on if you'd like a direct line to the developers.
