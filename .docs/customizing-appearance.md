# Customizing the Chat Widget Appearance

All appearance settings for Super Speedy Chat live in the **WordPress Customizer**, not the plugin settings page. This gives you a live preview of every change before you publish.

To access them, go to **Appearance > Customize > Super Speedy Chat** in your WordPress admin.

> **Note:** The plugin settings page (under the Super Speedy Chat admin menu) handles behaviour, email notifications, and other functional options. Visual customization is handled entirely through the Customizer.

## Available Settings

### Chat Header Image

Upload an image to display in the chat widget header bar. A recommended height of **40px** works best to keep the header compact.

### Chat Window Title

The text displayed in the chat header. Defaults to **"Chat"**. Change this to match your brand or use case (e.g. "Support", "Ask us anything").

### Primary Color

Controls the main accent color used for the **chat bubble trigger button** and other accent elements throughout the widget. Set this to match your site's brand color.

### Header Background Color

Sets the background color of the chat widget header bar. This is independent of the primary color, so you can use a contrasting or complementary shade.

### Visitor Message Color

Controls the background color of message bubbles sent by the visitor. Useful for distinguishing visitor messages from agent replies.

### Bubble Position

Choose where the chat trigger button appears on screen:

- **Bottom-right** (default)
- **Bottom-left**

### Trigger Icon

Select the icon shown on the floating chat trigger button. You can choose from three built-in options:

- Chat bubble
- Speech bubble
- Headset

Alternatively, upload a **custom image** to use as the trigger icon.

## Tips

- Use the Customizer's live preview to test color combinations before publishing.
- Keep the header image small (40px height) to avoid pushing the chat content area down.
- If your site has a fixed footer or other bottom-positioned elements, choose the bubble position that avoids overlap.
